# Dataset Scope

## 1. Target Variable

The primary target variable is lattice thermal conductivity (κ) of crystalline materials under pressure.

## 2. Units

Thermal conductivity will be expressed in W m⁻¹ K⁻¹.

## 3. Pressure Range

The dataset will include pressures ranging approximately from ambient pressure up to ~100 GPa when available.

## 4. Temperature Conditions

The initial focus will be room temperature (~300 K) thermal conductivity when possible.

## 5. Material Type

The dataset will focus on crystalline solids, including simple crystals, oxides, semiconductors, and other well-characterized materials studied under pressure.

## 6. Data Sources

Possible data sources include:
- published experimental literature
- computational studies using phonon Boltzmann transport calculations
- high-pressure materials studies

## 7. Required Input Features

- material composition
- crystal structure information
- pressure
- thermal conductivity value

## 8. Optional Advanced Features

Possible additional features include:
- density
- atomic mass statistics
- bonding characteristics
- phonon-related descriptors
